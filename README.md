# Budget Friendly Reels 🎬

Budget Friendly Reels is a premium web application for cinematic visual storytelling and professional reel production services. Built with a focus on high-impact visual aesthetics and lightning-fast performance.

## 🚀 Features

- **Premium UI/UX**: Modern "Glassmorphism" aesthetic with professional "Slate & Royal Indigo" theme.
- **Extreme Performance**:
  - **Hover-to-Play**: Background videos only play on interaction to save system resources.
  - **Lazy Load Architecture**: Components and images load only when needed.
  - **Hardware Acceleration**: Smooth scrolls and transitions optimized for GPU.
- **Interactive Portfolio**: Integrated high-quality video showcase with cinematic modal previews.
- **Service Mastery**: Highlighted expertise in Branding, E-commerce, Events, and more.
- **WhatsApp Optimized**: Native WhatsApp integration for instant local expert consultation across multiple cities (Vizag, Vijayawada, Hyderabad).
- **Leadership Spotlight**: Dedicated section for the founder and creative vision.

## 🛠️ Tech Stack

- **Core**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **UI Base**: Optimized Shadcn/UI components
- **Deployment**: Git LFS optimized for high-res assets

## 📦 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```
